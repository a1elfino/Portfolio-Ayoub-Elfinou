Projet de fin d’études

Réalisation d’un robot dessinateur

Il existe deux types de programmes à utiliser sur l’ordinateur : un pour créer le code d'une conception graphique donnée. Et un deuxième programme pour envoyer le code qui vient d'être créé afin que le traceur le dessine sur papier. Pour la première partie, nous a utilisé le programme de dessin vectoriel Inkscape. Pour la deuxième partie, nous utilisons le programme Java qui nous permet de charger le fichier créé avec Inkscape avant de l'envoyer au traceur.
